Entorno backend del proyecto final de Jesús Blázquez Durán: MATCH-ON! 
---

*¿Qué es MATCH-ON!* 

MATCH-ON! es una aplicación web dirigida a jugadores y organizadores de torneos **presenciales** y **online**.

Nuestro objetivo es ofrecer un servicio digitalizado de gestión y participación en torneos online con el objetivo de mejorar y facilitar el desarrollo de los mismos.
